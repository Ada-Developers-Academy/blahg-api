Blahg API

Installation

First install MongoDB

brew update
brew install mongodb
ln -sfv /usr/local/opt/mongodb/*.plist ~/Library/LaunchAgents
launchctl load ~/Library/LaunchAgents/homebrew.mxcl.mongodb.plist
# Check if installation was successful 
# If an interactive mongo shell opens, then it is 
mongo
exit

Next, clone the app and run it locally:

git clone git@github.com:Ada-Developers-Academy/blahg-api.git
cd blahg-api
bundle
rails server

Deploy

heroku create <name-of-app>
heroku addons:add mongolab:sandbox

Usage

GET /posts

Returns an array of Posts

[
  {
    "content": "Lorem Ipsum",
    "date": "2015-01-07T00:00:00.000Z",
    "id": "54ad8bba6f69fd0002000001",
    "tag_ids": [],
    "title": "Test"
  }
]
GET /posts/:id

Returns a Post with associated Tags

{
  "id": "54ac150d7115a44ac9000001",
  "tags": [
    {
      "id": "54ad73232781456703000001",
      "name": "yum"
    }
  ]
}
POST /posts

Creates a Post Accepted values:

Key Type
post[title] String
post[content] String
post[date] Time
DELETE /posts/:id

Deletes a Post

POST /posts/:post_id/tags

Find or creates a tag while adding it to a Post

Key Type
tag[name] String
DELETE /posts/:post_id/tags/:id

Removes a Tag from a Post

GET /tags

Returns all Tags

[
  {
    "id": "54ad8d8d6f69fd0002000003",
    "name": "Yum"
  }
]
GET /tags/:id

Returns a Tag

 
{
  "id": "54ad8d8d6f69fd0002000003",
  "name": "Yum"
}
POST /tags

Creates a Tag

DELETE /tags/:id

Deletes a Tag
