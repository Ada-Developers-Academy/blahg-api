Blahg API

Installation

First install MongoDB

brew update
brew install mongodb
ln -sfv /usr/local/opt/mongodb/*.plist ~/Library/LaunchAgents
launchctl load ~/Library/LaunchAgents/homebrew.mxcl.mongodb.plist
# Check if installation was successful 
# If an interactive mongo shell opens, then it is 
mongo
exit

Next, clone the app and run it locally:

git clone git@github.com:Ada-Developers-Academy/blahg-api.git
cd blahg-api
bundle
rails server

Deploy

heroku create <name-of-app>
heroku addons:add mongolab:sandbox

Usage

GET /posts Returns an array of Posts

[
  {
    "content": "Lorem Ipsum",
    "date": "2015-01-07T00:00:00.000Z",
    "id": "54ad8bba6f69fd0002000001",
    "tag_ids": [],
    "title": "Test"
  }
]

POST /posts Creates a Post

Accepted values:

Key Type
post[title] String
post[content] String
post[date] Time

DELETE /posts/:id Deletes a Post

POST /posts/:post_id/tagsFind or creates a tag while adding it to a Post

Key Type
tag[name] String

DELETE /posts/:post_id/tags/:idRemoves a Tag from a Post

GET /tagsReturns all Tags

[
  {
    "id": "54ad8d8d6f69fd0002000003",
    "name": "Yum"
  }
]

POST /tagsCreates a Tag

DELETE /tags/:idDeletes a Tag
